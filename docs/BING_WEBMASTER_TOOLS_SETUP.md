# Bing Webmaster Tools setup

Add `buyfootball.store` with a business-owned Microsoft account, import the verified Search Console property or use Bing’s supported verification method, and submit `https://buyfootball.store/sitemap.xml`. Review crawl information, index coverage, site scan findings and URL inspection after approved releases.

Keep verification tokens in public environment variables only when the provider expects a public meta token; never commit account credentials. IndexNow is not implemented because GitHub Pages has no secure server-side key notification workflow in this project. Reconsider it only with an approved, secure deployment process.
