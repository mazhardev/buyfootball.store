# Internal linking map

| Source | Required destinations | Useful anchor themes |
|---|---|---|
| Home | categories, shop, custom, wholesale, factory, quality, shipping, FAQ | product format; factory process; custom project |
| Shop | every visible product and category | exact product/category name |
| Category | products in that category, buying guide, shipping, wholesale | selection guide; shipping guidance; bulk requirement |
| Product | parent category, related products, relevant guide, custom/wholesale, shipping/returns | exact product; compare; care; quantity |
| Guide | relevant categories/products, related guides, custom/wholesale when useful | descriptive question/answer anchors |
| Country | categories, custom, private label, wholesale, shipping | market terminology without “local stock” claims |
| Factory/quality | custom, private label, wholesale, contact | specification; documentation; inquiry |
| Footer | core commerce, company, support and legal pages | concise route names |

Every indexable content page should receive at least one contextual link beyond global navigation. Avoid orphaned pages, excessive exact-match repetition and links to noindex market/demo content from high-authority editorial pages unless needed for review. `npm run seo:validate-links` checks exported paths and fragments.
