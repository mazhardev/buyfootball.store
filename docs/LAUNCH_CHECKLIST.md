# Launch checklist

## Business and content

- [ ] Legal business name, address, email, WhatsApp and social profiles verified
- [ ] All demonstration products replaced or signed off line by line
- [ ] Prices, comparison prices, sizes, colors, specifications, stock and MOQ verified
- [ ] Certification/testing claims supported by retained documentation
- [x] Sample testimonials removed; decorative stars remain aria-hidden and are not rating data
- [ ] Real product, factory, packaging, QC and customization photography installed
- [ ] Copyright/trademark rights checked for every name, logo, image and artwork
- [ ] Factory, quality, shipping and article content reviewed by operational owners

## Commerce

- [ ] Cart add/update/remove/clear tested across supported browsers and mobile
- [ ] Every product accepts only valid size/color combinations
- [ ] USD source price and EUR/GBP/CAD estimates checked
- [ ] WhatsApp and email order messages tested with real destination contacts
- [ ] External form endpoint tested, secured and disclosed—or left disabled
- [ ] Every hosted payment link verified—or left absent as “Request order”
- [ ] Shipping, duty, tax, production and returns expectations approved
- [ ] Wholesale/custom inquiry received and actionable end to end

## Legal and privacy

- [ ] Terms, privacy, cookies and returns reviewed by qualified advisers
- [ ] Analytics consent approach approved for target markets
- [ ] Data retention and customer-rights process documented
- [ ] External provider privacy and payment terms reviewed

## Technical and accessibility

- [ ] `npm run lint`, `npm run typecheck`, `npm run build` and `npm run seo:validate` pass
- [ ] Export includes index, 404, products, articles, countries, sitemap and robots
- [ ] Keyboard navigation, focus, Escape/overlay dialogs and mobile menu tested
- [ ] Labels, errors, headings, alt text, contrast and reduced motion reviewed
- [ ] Responsive layouts tested at small, medium and large widths
- [ ] No broken internal links or mixed-content resource
- [ ] Lighthouse/PageSpeed reviewed on the deployed production domain

## Search and deployment

- [ ] GitHub Pages source set to GitHub Actions
- [ ] `buyfootball.store` DNS, Pages custom domain and HTTPS verified
- [ ] Google Search Console domain property verified
- [ ] Sitemap submitted to Google and Bing
- [ ] Structured data tested without review warnings from sample content
- [ ] Country pages checked for unique value and no local-stock claim
- [ ] Analytics enabled only after privacy sign-off

## Post-launch

- [ ] Place a real low-risk inquiry through each channel
- [ ] Monitor Actions, uptime, 404s, indexing and contact delivery
- [ ] Assign owners for catalogue, rates, policies and content review cadence
- [ ] Plan backend migration before promising live inventory, tax, shipping or payment
